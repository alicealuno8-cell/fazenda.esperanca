# Sistema Fazenda Esperança

Projeto desenvolvido em Java para gerenciamento da colheita de café.

## Funcionalidades

- Cadastro de funcionários
- Cadastro de talhões
- Cadastro de tratores
- Registro de lançamentos
- Relatórios de produção

## Tecnologias Utilizadas

- Java
- ArrayList
- Scanner
- GitHub
